Engineering Triumph

The Transformers.js Preprocessor Solution

NIMBUS CORE
Primary Challenge: Preprocessor Serialization Bug

Underlying code inside the standard Hugging Face Transformers.js high-level pipeline('image-to-text') API fails to serialize raw image tensors and text prompts together for modern Gemma VLM architectures, throwing empty inputs or structural shape exceptions.

Our Solution: We bypassed the broken high-level pipeline altogether. By manually writing lower-level bindings, we orchestrate the input flow directly.

01 / Raw Extraction

Canvas programmatically extracts balanced visual frames from the video timeline.

02 / AutoProcessor

Processor reads raw pixel matrices and compiles them directly into normalized tensors.

03 / Message Injection

Manually injects matched placeholder objects into the tokenized message payload array.

04 / Direct Execution

Feeds combined tensors directly to the Gemma 4 E2B IT .generate() model method.
